# Wallet / Payment Backend System

## Description
A backend system that allows users to create accounts, maintain wallet balances,
and perform peer-to-peer money transfers with proper consistency and transaction tracking.

## Features
- User signup and login
- Wallet creation for each user
- Add money to wallet
- Peer-to-peer wallet transfers
- Transaction history with pagination

## Tech Stack
- Backend: Node.js + Express
- Database: PostgreSQL
- Authentication: JWT
